## Sobre EMK

Descripción del proyecto: El proyecto EMK permite gestionar y registrar ingresos y egresos de la empresa EMK. Es útil debido a que permite una noción constante sobre ingresos y egresos de los usuarios.

Instrucciones de instalación: Primero se ha de instalar las dependencias PHP (composer install), las dependencias de node (npm install), ahora se crea una copia del env (copy .env.example .env), ahora se crea la llave (php artisan key:generate), se ejecuta el entorno de desarrollo (npm run dev) y finalmente se despliega el proyecto (php artisan serve)

Ejemplos de uso: Si durante el mes de noviembre los empresarios de EMK realizan unas compras dentro del aplicativo esto se registrara y permitira buscarlas en cualquier momento con sus respectivos detalles, ademas de poder generar de ello.
